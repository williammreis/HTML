Entendendo Comunicação Client x Server

(Voltar)

Ao decorrer desse curso, aprendir e tirei dúvidas sobre o Client x Server. "Entendendo Comunicação Cliente-Servidor" trata-se de um conceito fundamental na computação e nas redes de computadores. Neste modelo, há duas entidades principais: o cliente e o servidor. O cliente é geralmente um dispositivo ou programa que solicita serviços ou recursos, enquanto o servidor é um dispositivo ou programa que fornece esses serviços ou recursos.

A comunicação entre cliente e servidor segue um padrão de "solicitação e resposta". O cliente envia uma solicitação ao servidor, especificando o que precisa (por exemplo, acessar um arquivo, obter dados de um banco de dados, etc.). O servidor recebe a solicitação, processa-a e, em seguida, envia uma resposta de volta ao cliente, contendo os dados solicitados ou confirmando a realização da ação requisitada.

Tags Aprendidas

Algumas das Tags que Aprendi no HTML:

(Voltar)
  • HTML - Elemento raiz de uma página HTML.
  • HEAD - Elemento que contém informações sobre a página HTML.
  • TITLE - Especifica o título da página HTML (que mostra na barra de título do navegador ou na guia de página).
  • BODY - É o corpo do documento HTML.
  • H1 a H6 - Definem títulos e tamanho dos títulos.
  • P - Define um parágrafo.
  • STRONG - Deixa texto em negrito.
  • I - Deixa o texto em itálico.
  • MARK - Faz uma marcação importante no texto.
  • SMALL - Texto menor.
  • SUB - Texto subscrito.
  • SUP - Texto sobrescrito.
  • U - Usada para sublinhar o Texto.
  • OL - Lista ordenada
  • UL - Lista não ordenada
  • LI - Representa um item que faz parte de uma lista.
  • HR - Representa uma linha temática.
  • A - Vincula links internos e externos.
  • BLOCKQUOTE - Indica que o texto incluído é uma longa citação.
  • NAV - Links de navegação de uma página que aponta para outras páginas ou para outras áreas da página.
  • DIV - Cria uma para Divisão para agrupar conteúdos.
